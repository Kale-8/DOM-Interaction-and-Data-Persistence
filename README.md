<h1 align="center">Interactive User Interface with Data Persistence</h1>

## 📝 Overview

This interactive project allows you to:

- Capture and store a user's name and age using **Local Storage**
- Display stored data on page load
- Track how many times the user has interacted during the current session using **Session Storage**
- Clear saved data
- Use **AlertifyJS** for real-time validation and success feedback

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- [AlertifyJS](https://alertifyjs.com/) (for beautiful notifications)

## 💻 How to Use

1. Open `index.html` in your browser
2. Enter a name and a valid age, then click **Save Data**
3. Reload the page to confirm data persistence
4. Click **Clear Data** to remove saved info
